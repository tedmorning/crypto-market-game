# Oh Jeez Man

### Install
```sh
git clone https://github.com/justinjung04/universal-boilerplate.git
cd universal-boilerplate
npm install
```

### Run the app from a dev server
```sh
npm start
```

### Build the app
```sh
npm run build
```

### Host the app from a server (app must have been built)
```sh
npm run serve
```
